# 如何使用

## 安装脚手架

```shell
npm install @addcn/core -g
```



## 初始化脚手架

```shell
addcn-cli init
```



## 使用基本手册



### 创建项目

```shell
addcn-cli create <app-name>
```



### 克隆一个线上仓库至本地

```shell
addc-cli clone <url> <app-name>
```



### 添加一个新模板至本地缓存

```shell
addcn-cli install <url>
```



### 部署项目

```shell
addcn-cli publish [options]
```



## 使用帮助

```shell
addcn-cli --help
```
